# Changelog

## 2.0.2

- Redesigned Heavy grid as CSS tree lines with continuous vertical branches.
- Heavy grid branch hover now highlights related vertical and horizontal line segments together.
- Added a frame option under Post excerpt - text.
- Text post excerpts can now be shown either without a frame or with a compact framed bubble.
- Heavy grid now keeps visual branches for posts whose original parent is no longer visible in the current topic after moderation.

## 2.0.1

- Preserved line breaks in text post excerpts.
- Removed extra blank lines between real text excerpt lines.
- Clarified that TreeK can normally be updated by installing the new package over the existing one.

## 2.0

- Split TreeK into Free and Pro editions.
- Added Free and Pro package builds.
- Added Pro preview items in the Free settings window.
- Added forum-view settings.
- Added settings save/restore for Pro.
- Improved reply form behavior with Comfort form header and subject suffixes.
- Added package metadata for the 2.0 release line.

## Earlier Prototype

The initial repository history contained an early prototype. The public
repository was reset to documentation and release information before the first
public TreeK Free release.
