# JED Description Draft

TreeK adds a post tree to Kunena topics. It makes long forum discussions easier
to read by showing the reply structure of a topic and helping users jump between
related posts.

## Short Description

TreeK adds a structured post tree to Kunena Forum topics.

## Main Features

- Open a post tree from Kunena topics.
- Display post hierarchy.
- Highlight the latest and active posts.
- Configure tree display in a settings window.
- Show post text teaser and tooltip.
- Show date, time, and post index.
- Export the tree as text.
- Navigate from the tree back to the Kunena topic.
- Parent-post navigation in the forum view.

## Pro Features

TreeK Pro adds screenshot teaser, flat view, comfort symbols, BBCode/HTML export,
topic monitoring, draggable settings, save and restore settings, subject suffixes, collapsed
attachments, and inline action buttons.
